dfnGen - C++ Generation Code  
#################################

dfnGen creates the discrete fracture networks using the feature rejection algorithm for meshing (FRAM). Fractures can be created stochastically or as deterministic features. 

The detailed description of FRAM and the implemented methodology is in `\J. D. Hyman, C. W. Gable, S. L. Painter, and N. Makedonska. Conforming Delaunay triangulation of stochastically generated three dimensional discrete fracture networks: A feature rejection algorithm for meshing strategy. SIAM J. Sci. Comput., 36(4):A1871â€“A1894, 2014 <https://epubs.siam.org/doi/abs/10.1137/130942541>`_.
